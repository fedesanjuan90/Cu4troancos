# Proyecto Final React - Cu4troancos 🍻🛒

Este es el proyecto final del curso de React JS de Coderhouse, desarrollado por Federico Sanjuan.

## 🧩 Funcionalidades

- ✅ Navegación con React Router
- ✅ Catálogo de productos con filtrado por categorías
- ✅ Vista de detalle por producto
- ✅ Carrito de compras con funcionalidad completa
- ✅ Formulario de checkout con generación de orden en Firebase
- ✅ Formulario de contacto funcional (EmailJS)
- ✅ Links a WhatsApp, correo, Instagram y ubicación en Google Maps
- ✅ Diseño responsive

## 🛠 Tecnologías utilizadas

- React + Vite
- Firebase (Firestore)
- EmailJS
- React Router DOM
- CSS3
